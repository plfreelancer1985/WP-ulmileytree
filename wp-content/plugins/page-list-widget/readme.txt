== Description ==

This is a widget plugin. This widget will display a list of posts/pages.

* This widget displays posts/pages.
* Can change sort order.
* Can display/hide updated date.
* Can include some pages from all pages.
* Can exclude some pages from the list.
* If you want to translate this plugin, you put '*.po' and '*.mo' in 'languages' directory. Japanese language file is included in this plugin.

== Installation ==

1. Upload this directory to the 'wp-content/plugins/' directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Open 'Widgets' menu, add the 'Post/Page list' widget into the sidebar and choose your options.
